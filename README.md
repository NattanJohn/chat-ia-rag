# Chat de IA com RAG

Projeto inicial de um Chat de Inteligência Artificial utilizando a abordagem RAG (Retrieval-Augmented Generation).

## Estrutura do projeto

- `frontend/` - Aplicação React + TypeScript + Vite
- `backend/` - API Node.js + TypeScript + Express
- `.env` - Arquivo para variáveis de ambiente (não incluído no repositório)

## Como rodar localmente

### Backend
```bash
cd backend
npm install
npm run dev

#### Frontend
cd frontend
npm install
npm run dev
